# ARCHIVOS DE AUDIO REQUERIDOS

## Música por Ambientación

Cada ambientación requiere 3 archivos de música de fondo en formato MP3:

### wwii/
- march_1.mp3 - Marcha militar
- battle_1.mp3 - Música de batalla
- victory_1.mp3 - Música de victoria

### antigua/
- lyre_1.mp3 - Música con lira griega
- flute_1.mp3 - Flauta antigua
- drum_1.mp3 - Tambores antiguos

### videojuegos/
- chiptune_1.mp3 - Música estilo 8-bit
- arcade_1.mp3 - Música arcade
- synth_1.mp3 - Synthwave gaming

### peliculas/
- orchestra_1.mp3 - Orquesta cinematográfica
- action_1.mp3 - Música de acción
- epic_1.mp3 - Música épica

### anime/
- epic_1.mp3 - Opening épico
- battle_1.mp3 - Música de batalla anime
- adventure_1.mp3 - Música de aventura

### comics/
- hero_1.mp3 - Tema de héroe
- action_1.mp3 - Acción cómic
- energy_1.mp3 - Energía superheroica

### literatura/
- piano_1.mp3 - Piano suave
- violin_1.mp3 - Violín clásico
- library_1.mp3 - Ambiente de biblioteca

### religiones/
- choir_1.mp3 - Coro religioso
- organ_1.mp3 - Órgano de iglesia
- spiritual_1.mp3 - Música espiritual

---

## Efectos de Sonido (SFX)

Cada ambientación requiere 7 efectos de sonido:

- sfx_button.mp3 - Clic de botón
- sfx_select.mp3 - Selección
- sfx_move.mp3 - Movimiento de tropas
- sfx_combat.mp3 - Sonido de combate
- sfx_victory.mp3 - Fanfarria de victoria
- sfx_defeat.mp3 - Sonido de derrota
- sfx_unlock.mp3 - Desbloqueo/logro

---

## NOTA IMPORTANTE

Si no tienes los archivos de audio reales, el sistema utilizará automáticamente
la Web Audio API para generar sonidos sintetizados como fallback.

Los archivos MP3 deben colocarse en sus respectivas carpetas:
- /workspace/audio/wwii/
- /workspace/audio/antigua/
- /workspace/audio/videojuegos/
- /workspace/audio/peliculas/
- /workspace/audio/anime/
- /workspace/audio/comics/
- /workspace/audio/literatura/
- /workspace/audio/religiones/
